$\lim_{ x \to 0 }\frac{\arctan x}{x} =1$
Proof:
$\lim_{ x \to 0 }\frac{\arctan x}{x} = \lim_{ t \to 0 }\frac{t}{\tan t}=1$ (по теореме о пределе композиции функций)
